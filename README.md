# diabetes-risk-prediction
A machine learning-based system that predicts diabetes risk using healthcare data. It includes data preprocessing, model training, and Explainable AI techniques to improve transparency and understanding of predictions.

# Diabetes Risk Prediction System

## 📌 Overview
This project predicts diabetes risk using machine learning.

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn

## 🚀 Features
- Data preprocessing
- Model training
- Explainable AI (XAI)

## ▶️ How to Run
1. Install required libraries
2. Run the main file

## 📊 Output
Predicts whether a person is diabetic or not
